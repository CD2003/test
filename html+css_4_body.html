画像を表示する

<img src = "★″ alt = "♦″>

<img>タグを使用して、文書に画像を埋め込むことができる。


 属性  指定方法 説明
★画像ファイル名(URL)画像のURL(ファイル名)を指定する。
♦テキスト(URL)画像の代わりのテキストを指定する。
ブラウザが画像に未対応、またユーザが画像の非表示設定をしている場合には、
alt属性でしたいしたテキストが表示される。

<body>
    <div>
        <img src = "2Cats.jpg″ alt = "コロンとレオ″>
    </div>
    <div>
        <img src = "2Cats2.jpg″ alt = "コロンとレオ″>        <!-- alt属性表示用 -->
    </div>
</body>

<img src = "★″ width = "♦″ height = "▴″>

画像の表示サイズを変更したい場合には、width属性、height属性を使用する。


 属性  指定方法 説明

  &emspG; </div>

画像の枠線を指定する

border-width : ★枠線の幅
border-color : ♦枠線の色
border-style : ▴枠線の種類
border : ★ ♦ ▴枠線の一括指定

画像の枠線は、CSSのボーダー関連のプロパティで指定する。
border-width、border-color、border-styleプロパティは、それぞれ上下左右のボーダー(枠線)の幅、色、種類を一括して指定するプロパティである。いずれも値が1つだけのときは上下左右に同じ値が適用されるが、2∼4個の値を半角スペースで区切って並べることで、値の数によって下記のように適用される。

★上下左右同じ
★ ★上下、左右
★ ★ ★上、左右、下
★ ★ ★ ★上、右、下、左

なお、border-widthプロパティを有効にするには、border-styleプロパティで枠線の種類をしておくこと。

枠線の幅

幅指定は以下を指定する。

thin細い線細い線
midium中くらいの線中くらいの線
thick太い線太い線
実数値+単位数値に単位を付けてボーダー幅を指定10px

枠線の色

枠線の色は、border-colorプロパテティで指定する。

色RGB値もしくはキーワード(色名)で指定
transparent透明を指定(ボーダーの幅は保持)

枠線の種類

枠線の種類は、border-styleプロパティで指定する。

dotted点線
dashed破線
solid実線
double二重線
groove凹んだように見える線
ridge浮き上がったように見える線
inset内側が凹んだように見える線
outset内側が浮き上がったように見える線
none、hiddenボーダー非表示

一括して指定する

ボーダーの一括指定は、borderプロパティを指定する。

 属性  指定方法 説明
★"枠線の幅″項を参照枠線の幅を指定する。
♦"枠線の種類″項を参照枠線の種類を指定する。
▴"枠線の色″項を参照枠線の色を指定する。

[ html ]
<body>
    <div>
        <img src = "2Cats.jpg″ id = "sample1″ width = "185″ height = "200″ alt ="″>
        <img src = "2Cats.jpg″ id = "sample2″ width = "185″ height = "200″ alt ="″>
    </div>
</body>

[ css ]
img {
    margin:10px;
}

img.sample3-1
{
    border-width:2px;
    border-color:♯ff6600;
    border-style:solid;
}

img.sample3-2 {
    border:4px ♯009966 dotted;
}

画像とテキストとの並び方を指定する

vertical-align : ★

画像はインライン要素であるため、通常はテキストと一緒に1つの行の中に配置されます。その場合の画像とテキストの並び方には、vertical-alignプロパティで指定する。
vertical-alignプロパティは、指定要素が表示される行の中で縦位置を指定する。


値(★)の指定方法

 属性  指定方法 説明
★baseline要素ボックスのベースラインを親要素ボックスのベースラインに揃える。
top要素ボックスの上を親要素ボックスの上に揃える。
middle要素ボックスの中心を親要素ボックスの小文字xの中心に揃える。
bottom要素ボックスの下を親要素ボックスの下に揃える。
text-top要素の上辺を親要素のフォントの上辺に揃える。
text-bottom要素の下辺を親要素のフォントの下辺に揃える。
subその要素を下付き文字にする。
superその要素を上付き文字にする。
実数値+単位数値に単位を付けて要素をどれだけ上げるか(マイナス値の場合は下げる)を指定する。
数値+%line-heightプロパティの値に対する割合でその要素をどれだけ上げる(マイナス時は下げる)かを指定する。

< html >
<body>
    <p>

    </p>
    <p>
        <img src = "2Cats.jpg″ class = "sample4-2″ width = "200″ height = "200″ alt = "″>

画像にテキストを回り込ませる

float : ★

画像にテキストを回り困る場合には、CSSのfloatプロパティを使用する。
floatプロパティは対象の要素ボックス領域を左、あるいは右に寄せ、あとに続く内容をその反対側に回り込ませる。


値(★)の指定方法

 属性  指定方法 説明
★left要素ボックスを左側に配置し、あとに続く内容を右側に回り込ませる。
right要素ボックスを右側に配置し、あとに続く内容を左側に回り込ませる。

< html >
<body>
    <p>
        <img src = "2Cats.jpg″ alt = "コロンとレオ″>コロンとレオは姉弟で、5年前に拾いました。コロンは長毛の白猫で、レオは短毛の白黒ハチワレです。なお、妹猫にみかんがいます。みかんは短毛のキジ白でコロン、レオとは血の繋がりはありませんが、仲良く暮らしております。
    </p>
</body>

< CSS >
img
{
    float:left;
}

コロンとレオは姉弟で、5年前に拾いました。コロンは長毛の白猫で、レオは短毛の白黒ハチワレです。なお、妹猫にみかんがいます。みかんは短毛のキジ白でコロン、レオとは血の繋がりはありませんが、仲良く暮らしております。

イメージマップを作成する

<img src = "★″ usemap = "♯♦″ alt = "▾″>
<map name = "♦″>∼</map>
<area shape = "▴″ coords = "&circle;″ href = "□″ alt = "▾″>

イメージマップ機能を利用すると、画像の特定の領域にリンクを設定することができる。ここで解説するクライアントサイドイメージマップは全処理をブラウザ側で実行するものであり、HTMLタグで作成できる。


<img>と<map>

<map name = "♦″>∼</map>で、画像のクライアントサイド・イメージマップの定義と、そのイメージマップの名前を定義する。
<img src = "★″ usemap = "♯♦″>はusemap属性に<map>タグのnameゾック性で指定した名前を指定して、画像とイメージマップの関連付けを行う。

 属性  指定方法 説明
★画像ファイル名(URL)画像ファイルのURLを指定する。
♦マップ名マップ名を指定する。。

<area>

リンク領域は<area>タグで次の属性を用いて指定する。
shape属性
 画像上でマップとして定義される領域の形を指定する。

 属性  指定方法 説明
▴defalult全体
rect四角形
circle円
poly多角形

coord属性
 領域の座標を指定する。ピクセルで指定し、各座標値はカンマ(,)で区切る。
 属性  指定方法 説明
&circle;左上のX座標,左上のY座標,右下のX座標,右下のY座標rectの場合
中心のX座標,中心のY座標,半径円の場合
全ての角の座標を(X座標,Y座標)の順で指定し、最後に最初の座標を入力多角形の場合

 属性  指定方法 説明
★画像ファイル名(URL)画像ファイルのURLを指定する。。
♦マップ名マップ名を指定する。。
♦マップ名マップ名を指定する。。

auto自動で調整する。
