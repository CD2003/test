画像を表示する

<img src = "★″ alt = "♦″>

<img>タグを使用して、文書に画像を埋め込むことができる。


 属性  指定方法 説明
★画像ファイル名(URL)画像のURL(ファイル名)を指定する。
♦テキスト(URL)画像の代わりのテキストを指定する。
ブラウザが画像に未対応、またユーザが画像の非表示設定をしている場合には、
alt属性でしたいしたテキストが表示される。

<body>
    <div>
        <img src = "2Cats.jpg″ alt = "コロンとレオ″>
    </div>
    <div>
        <img src = "2Cats2.jpg″ alt = "コロンとレオ″>        <!-- alt属性表示用 -->
    </div>
</body>

画像の表示サイズを指定する

<img src = "★″ width = "♦″ height = "▴″>

画像の表示サイズを変更したい場合には、width属性、height属性を使用する。


 属性  指定方法 説明
★画像ファイル名(URL)画像のURL(ファイル名)を指定する。
♦幅(ピクセル数、あるいは%)画像の幅サイズを指定する。
%指定時は、相対的になる。
▴高さ(ピクセル数、あるいは%)画像の高さサイズを指定する。
%指定時は、相対的になる。

<body>
    <div>
        <img src = "2Cats.jpg″ width = "185″ height = "200″ alt ="″>
        <img src = "2Cats.jpg″ width = "100″ height = "300″ alt ="″>
        <img src = "2Cats.jpg″ width = "25%″ height = "20%″ alt ="″>
    </div>
</body>
