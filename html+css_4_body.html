画像を表示する

<img src = "★″ alt = "♦″>

<img>タグを使用して、文書に画像を埋め込むことができる。


 属性  指定方法 説明
★画像ファイル名(URL)画像のURL(ファイル名)を指定する。
♦テキスト(URL)画像の代わりのテキストを指定する。
ブラウザが画像に未対応、またユーザが画像の非表示設定をしている場合には、
alt属性でしたいしたテキストが表示される。

<body>
    <div>
        <img src = "2Cats.jpg″ alt = "コロンとレオ″>
    </div>
    <div>
        <img src = "2Cats2.jpg″ alt = "コロンとレオ″>        <!-- alt属性表示用 -->
    </div>
</body>

画像の表示サイズを指定する

<img src = "★″ width = "♦″ height = "▴″>

画像の表示サイズを変更したい場合には、width属性、height属性を使用する。


 属性  指定方法 説明
★画像ファイル名(URL)画像のURL(ファイル名)を指定する。
♦幅(ピクセル数、あるいは%)画像の幅サイズを指定する。
%指定時は、相対的になる。
▴高さ(ピクセル数、あるいは%)画像の高さサイズを指定する。
%指定時は、相対的になる。

<body>
    <div>
        <img src = "2Cats.jpg″ width = "185″ height = "200″ alt ="″>
        <img src = "2Cats.jpg″ width = "100″ height = "300″ alt ="″>
        <img src = "2Cats.jpg″ width = "25%″ height = "20%″ alt ="″>
  &emspG; </div>
</body>

画像の枠線を指定する

border-width : ★枠線の幅
border-color : ♦枠線の色
border-style : ▴枠線の種類
border : ★ ♦ ▴枠線の一括指定

画像の枠線は、CSSのボーダー関連のプロパティで指定する。
border-width、border-color、border-styleプロパティは、それぞれ上下左右のボーダー(枠線)の幅、色、種類を一括して指定するプロパティである。いずれも値が1つだけのときは上下左右に同じ値が適用されるが、2∼4個の値を半角スペースで区切って並べることで、値の数によって下記のように適用される。

★上下左右同じ
★ ★上下、左右
★ ★ ★上、左右、下
★ ★ ★ ★上、右、下、左

なお、border-widthプロパティを有効にするには、border-styleプロパティで枠線の種類をしておくこと。

枠線の幅

幅指定は以下を指定する。

thin細い線細い線
midium中くらいの線中くらいの線
thick太い線太い線
実数値+単位数値に単位を付けてボーダー幅を指定10px

枠線の色

枠線の色は、border-colorプロパテティで指定する。

色RGB値もしくはキーワード(色名)で指定
transparent透明を指定(ボーダーの幅は保持)

枠線の種類

枠線の種類は、border-styleプロパティで指定する。

dotted点線
dashed破線
solid実線
double二重線
groove凹んだように見える線
ridge浮き上がったように見える線
inset内側が凹んだように見える線
outset内側が浮き上がったように見える線
none、hiddenボーダー非表示

一括して指定する

ボーダーの一括指定は、borderプロパティを指定する。

 属性  指定方法 説明
★"枠線の幅″項を参照枠線の幅を指定する。
♦"枠線の種類″項を参照枠線の種類を指定する。
▴"枠線の色″項を参照枠線の色を指定する。

[ html ]
<body>
    <div>
        <img src = "2Cats.jpg″ id = "sample1″ width = "185″ height = "200″ alt ="″>
        <img src = "2Cats.jpg″ id = "sample2″ width = "185″ height = "200″ alt ="″>
    </div>
</body>
