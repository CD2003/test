<a href = "♯★″>∼</a>
<a href = "♦♯★″>∼</a>
<▴ id href = "★″>∼</▴>

<a>タグとid属性で、指定位置に移動するリンクを作成できる。
同一ページ内での移動は、リンク先のタグに<▴ id = "★″>のようにid属性で名前を付与し、指定した名前をリンク元の<a href = "♯★″>∼</a>の値に指定する。
他のページの移動は、<a href = "♦♯★″>∼</a>のようにid属性を指定した名前(★)の前にリンク先のURL(♦)を追加する。

属性指定方法説明
★名前リンクに名前を作成する
♦URLリンク先のURLを入力する
▴要素名要素を入力する
