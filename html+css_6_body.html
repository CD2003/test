リストを作成する

<ul><li>∼</ul></li>

<ul>∼</ul>タグで、その範囲(∼)が順不同(項目の順序が重要でない)のリスト(箇条書きで記載)であることを示す。
<li>∼</li>タグでリストで表示する項目(一般的に、各リストの先頭に"˙″)を指定する。

<body>
    <p>
        好きな果物は以下です。
    </p>
    <ul>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ul>
</body>

◦出力結果

好きな果物は以下です。

  • 蜜柑
  • 林檎
  • 梨

番号付きのリストを作成する

<ol><li>∼</ol></li>

<ol>∼</ol>タグで、その範囲(∼)が番号付き(順番が重要)のリストであることを示す。
<li>∼</li>タグでリストで表示する項目(各リストの先頭に数字)が表示される。

<body>
    <p>
        好きな果物のベスト3は以下です。
    </p>
    <ol>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ol>
</body>

◦出力結果

好きな果物はのベスト3は以下です。

  1. 蜜柑
  2. 林檎
  3. 梨

リストのマークを変更する

list-style-type:★

リストの各項目の先頭に表示されるマークの種類を変更する。
なお、リスト番号の種類を変更する場合にもlist-style-typeプロパティを使用する。

 属性  指定方法    説明   
★disc黒丸(初期値)
circle白丸
square資格
noneマークなし

[html]
<body>
    <ul = "sample3-1″>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ul>

    <ul>
        <li class = "sample3-2″>蜜柑</li>
        <li class = "sample3-3″>林檎</li>
        <li class = "sample3-4″>梨</li>
    </ul>
</body>

[css]
ul.sample3-1
{
    list-style-type:none;
}

li.sample3-2
{
    list-style-type:disc;
}

li.sample3-3
{
    list-style-type:circle;
}

li.sample3-4
{
    list-style-type:square;
}

用語の定義をリスト表示する

<dl><dt>∼</dt><dd>∼</dd></dl>

<dl>∼</dl>でその範囲を定義リストに指定する。定義リストとは、定義したい用語と用語に対する解説文とで形成されたリストを指す。
<dt>∼</dt>に定義したい用語を、<dd>∼</dd>にdt要素で指定した要素に対する解説や定義を記述する。
<dt>と<dd>はセットで使用するが、<dl>∼</dl>間には複数並べることが可能である。

[html]
<body>
    <dl>
        <dt>蜜柑</dt>
        <dd>柑橘系の一種。外見色は橙色。</dd>
        <dt>富士</dt>
        <dd>林檎系の一種。外見色は赤色。</dd>
    </dl>
</body>

◦出力結果

蜜柑
柑橘系の一種。外見色は橙色。
林檎
林檎系の一種。外見色は赤色。
