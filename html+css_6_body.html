リストを作成する

<ul><li>∼</ul></li>

<ul>∼</ul>タグで、その範囲(∼)が順不同(項目の順序が重要でない)のリスト(箇条書きで記載)であることを示す。
<li>∼</li>タグでリストで表示する項目(一般的に、各リストの先頭に"˙″)を指定する。

<body>
    <p>
        好きな果物は以下です。
    </p>
    <ul>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ul>
</body>

◦出力結果

好きな果物は以下です。

  • 蜜柑
  • 林檎
  • 梨

番号付きのリストを作成する

<ol><li>∼</ol></li>

<ol>∼</ol>タグで、その範囲(∼)が番号付き(順番が重要)のリストであることを示す。
<li>∼</li>タグでリストで表示する項目(各リストの先頭に数字)が表示される。

<body>
    <p>
        好きな果物のベスト3は以下です。
    </p>
    <ol>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ol>
</body>

◦出力結果

好きな果物はのベスト3は以下です。

  1. 蜜柑
  2. 林檎
  3. 梨

リストのマークを変更する

list-style-type:★

リストの各項目の先頭に表示されるマークの種類を変更する。
なお、リスト番号の種類を変更する場合にもlist-style-typeプロパティを使用する。

 属性  指定方法    説明   
★disc黒丸(初期値)
circle白丸
square資格
noneマークなし

[html]
<body>
    <ul = "sample3-1″>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
    </ul>

    <ul>
        <li class = "sample3-2″>蜜柑</li>
        <li class = "sample3-3″>林檎</li>
        <li class = "sample3-4″>梨</li>
    </ul>
</body>

[css]
ul.sample3-1
{
    list-style-type:none;
}

li.sample3-2
{
    list-style-type:disc;
}

li.sample3-3
{
    list-style-type:circle;
}

li.sample3-4
{
    list-style-type:square;
}

◦出力結果

  • 蜜柑
  • 林檎
  • 梨
  • 蜜柑
  • 林檎
  • 梨

リストの番号の種類を変更する

list-style-type:★

リストの各項目の先頭に表示される番号の種類の変更にも、list-style-typeプロパティを使用する。

 属性  指定方法    説明   
★decimal10進数(1、2、3、…)
decimal-leading-zero0を頭に付けた10進数(01、02、03、…、98、99)
lower-roman小文字ローマ数字(ⅰ、ⅱ、ⅲ、…)
upper-roman大文字ローマ数字(Ⅰ、Ⅱ、Ⅲ、…)
lower-greek小文字ギリシア文字(α、β、γ、…)
upper-greek大文字ギリシア文字(Α、Β、Γ、…)
lower-latin小文字アルファベット(a、b、c、…)
lower-alpha
upper-latin大文字アルファベット(A、B、C、…)
upper-alpha
noneマークなし

[html]
<body>
    <ol = "sample4-1″>
        <li>蜜柑</li>
        <li>林檎</li>
        <li>梨</li>
        <li>桃</li>
    </ol>

    <ol>
        <li>蜜柑</li>
        <li>林檎</li>
        <li class = "sample4-2″>梨</li>
        <li class = "sample4-3″>桃</li>
    </ol>
</body>

[css]
ol.sample4-1
{
    list-style-type:lower-alpha;
}

li.sample4-2
{
    list-style-type:lower-roman;
}

li.sample4-3
{
    list-style-type:upper-roman;
}

  4. 桃
