テーブルを作る

< table>∼</table>
<tr>∼</tr>
<td>∼</td>

上記3つのタグで基本的なターブルを作成する。

<table>
<tr>ピーマン</td><td>緑色の</td><td>野菜</td></tr>
<tr>トマト</td><td>赤色の</td><td>野菜</td></tr>
<tr>蜜柑</td><td>オレンジ色の</td><td>果物</td></tr>
</table>

◦出力結果

ピーマン緑色の野菜
トマト赤色の野菜
蜜柑オレンジ色の果物

表の枠線を表示する

border–width:★
border–color:◆
border–style:▲
border:★◆▲

枠線を表示する

[HTML]
<table>

<tr>トマト</td><td>赤色の</td><td>野菜</td></tr>
<tr>蜜柑</td><td>オレンジ色の</td><td>果物</td></tr>
</table>

[CSS]

{
    border-width:2px
    border-color:gray
    border-style:solid

    border-width:4px
    border-color:gray

}

◦出力結果

ピーマン緑色の野菜
トマト赤色の野菜
蜜柑オレンジ色の果物

上下左右の枠線を個別に指定する

上下左右で枠線のステータスを変更したい場合、半角スペースで区切って書くことで変更できる。

★:上下左右同じ
★ ◆:★が上下、◆が左右
★ ◆ ▲:★が上、◆が左右、▲が下
★ ◆ ▲ ●:★が上、◆が右、▲が下、●が左

枠線の幅

幅指定は以下を使用する。

thin細い線
medium中くらいの線
thick太い線
実数値+単位10px(ただし、マイナス値は指定不可)

枠線の色

色指定は以下を使用する。

色green(RGB値も可)
transparent透明

枠線の種類

枠線の種類は以下。

dotted点線
dashed破線
solid実線
double二重線
groove凹んだように見える線
ridge浮き上がったように見える線
inset内側が凹んだように見える線
outset内側が浮き上がったように見える線
none、hidden枠線非表示
